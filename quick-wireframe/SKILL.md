Create a simple HTML/CSS mockup from the user's text description. Keep it lightweight.

## Process
1. Parse the app idea
2. Identify key screens/components  
3. Create single-page HTML mockup
4. Use basic clean styling

## Output
- Single HTML file with embedded CSS
- Simple navigation between sections
- Placeholder content showing app structure
