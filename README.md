## 📞 Skill Descriptions

### Product Management Skills

**pm-strategy**
Define the "why" behind product work. Establishes strategic context, validates problem/opportunity fit, and ensures all downstream decisions align with business objectives and market realities.

**pm-product-circle**
8-phase product development lifecycle that coordinates all PM skills from strategic conception through post-launch optimization. Maps phases, skill ownership, handoffs, and validation gates.

**pm-skill-template**
Reference template that defines the standard structure for all PM skills. Use to understand the format and consistency expected across all skills.

**tier2-design**
Drive user-centered design from problem definition through launch. Ensures products are intuitive, accessible, and delightful while translating business requirements into visual and interaction systems.

**tier2-tech-architecture**
Define technical foundations, system design, and feasibility validation. Ensures architectural decisions enable product goals, support scalability, and provide clear implementation guidance.

**tier2-qa**
Ensure product quality through shift-left testing starting Week 8, comprehensive test coverage, and continuous quality validation from design through launch.

**pm-execution**
Guide for tactical execution of project plans through sprint management, dependency coordination, and risk tracking. Focuses on day-to-day delivery execution.

**product-launch**
Plan and execute product launches and marketing campaigns. Focuses on GTM strategy execution, multi-phase launch coordination, quality gates, and post-launch optimization.

### Document Skills

**docx**
Create, read, edit, and format Word documents (.docx). Supports tables of contents, headings, page numbers, and professional formatting.

**pdf**
Work with PDF files: read, extract, merge, split, rotate, watermark, create forms, OCR, and more.

**xlsx**
Create and edit spreadsheets (.xlsx, .csv, .tsv). Support for formulas, formatting, charts, and data manipulation.

**internal-comms**
Write internal communications: status reports, leadership updates, newsletters, FAQs, incident reports, and project updates.

**doc-coauthoring**
Guide for co-authoring documentation, proposals, and technical specs with structured workflows and iteration.

### Design & Art Skills

**canvas-design**
Create beautiful visual art in PNG and PDF documents using design philosophy. For posters, artwork, and static designs.

**algorithmic-art**
Create algorithmic art using p5.js with seeded randomness and interactive parameter exploration.

**slack-gif-creator**
Create animated GIFs optimized for Slack with specific constraints, validation, and animation concepts.

**quick-wireframe**
Create lightweight wireframes and mockups from text descriptions for rapid prototyping.

**theme-factory**
Apply pre-set themes or generate custom themes with colors and fonts to artifacts.

**brand-guidelines**
Apply Anthropic's official brand colors and typography to artifacts.

### Frontend Development Skills

**frontend-design**
Create distinctive, production-grade frontend interfaces with high design quality. Avoids generic AI aesthetics.

**web-artifacts-builder**
Build elaborate React artifacts using modern frontend technologies (React, Tailwind CSS, shadcn/ui). For complex artifacts requiring state management.

**web-components**
HTML/CSS components for portfolio sites: navigation bars, project cards, timeline layouts, animations.

**webapp-testing**
Interact with and test local web applications using Playwright. Supports verification, debugging, screenshots, and browser logs.

### Presentation Skills

**pptx**
Create PowerPoint presentations and pitch decks. Supports slide creation, editing, templates, speaker notes, and comments.

### Developer Tools

**claude-api**
Build apps with Claude API and Anthropic SDK. For code importing `anthropic` or `@anthropic-ai/sdk`.

**mcp-builder**
Create MCP (Model Context Protocol) servers that enable LLMs to interact with external services through well-designed tools.

**skill-creator**
Create new skills, modify and improve existing skills, and measure skill performance with evaluations.

### Portfolio & Content Skills

**portfolio-management**
Manage, update, and deploy portfolio websites to GitHub Pages. Add projects, fix navigation, and maintain content.

**content-generator**
Write case studies, professional bios, LinkedIn copy, and impact bullet points for professional portfolios.
